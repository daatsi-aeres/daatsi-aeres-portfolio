Autonomous Quadruped Robot

A bio-inspired quadruped robot capable of navigating complex terrains using reinforcement learning and advanced locomotion algorithms.

ROS Python Gazebo Reinforcement Learning

Project Overview

Challenge

Traditional quadruped robots struggle with dynamic and uneven terrains, often requiring manual tuning of gait parameters for different environments. The challenge was to develop a system that could autonomously adapt its locomotion strategy to varying terrain conditions.

Solution

I developed a reinforcement learning framework that enables the robot to learn optimal gait parameters through trial and error in simulation. The system uses proprioceptive feedback and terrain perception to dynamically adjust its walking pattern, allowing it to traverse slopes, stairs, and rough terrain with minimal energy expenditure.

Implementation

The project was implemented using ROS for robot control and Gazebo for simulation. The reinforcement learning algorithm was trained using PyTorch, with the policy network taking sensor inputs (joint angles, IMU data, and terrain heightmap) and outputting optimal joint trajectories. The trained model was then deployed on the physical robot with a NVIDIA Jetson Xavier for real-time inference.

Key Features

  • Adaptive gait control for varied terrains
  • Energy-efficient locomotion
  • Self-righting capability
  • Obstacle negotiation up to 30cm

Technical Specs

  • NVIDIA Jetson Xavier NX
  • 12V 5000mAh LiPo battery
  • Dimensions: 60cm × 40cm × 30cm
  • 8 Dynamixel servos for actuation
