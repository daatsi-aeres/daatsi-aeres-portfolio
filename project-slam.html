3D SLAM for Mobile Robots

A robust Simultaneous Localization and Mapping system using LIDAR and visual odometry for autonomous indoor navigation in dynamic environments.

C++ ROS LIDAR OpenCV

Project Overview

Challenge

Autonomous mobile robots operating in indoor environments require accurate localization and mapping capabilities. Traditional 2D SLAM systems struggle with multi-floor environments and dynamic obstacles, leading to navigation failures.

Solution

I developed a 3D SLAM system that fuses data from a 360Â° LIDAR and stereo camera to create dense 3D maps while maintaining real-time performance. The system incorporates loop closure detection and dynamic object filtering to improve accuracy in changing environments.

Implementation

The system was implemented in C++ using ROS for communication between modules. The frontend processes sensor data to estimate odometry, while the backend performs graph optimization using g2o. A custom octree-based representation was developed for efficient 3D map storage and querying, allowing for real-time updates on resource-constrained hardware.

Key Features

  • Real-time 3D mapping at 10Hz
  • Dynamic object filtering
  • Multi-floor environment support
  • Loop closure detection

Technical Specs

  • Intel i7-8550U processor
  • Velodyne VLP-16 LIDAR
  • ZED stereo camera
  • 5cm map resolution
